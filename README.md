Short project to run a hangman game. I built this during an introduction to Python at a coding school.
The code is originally intended to run on Google Colab, though can be modified to run in other
Python 3 environments if the first import and all statements of ´output.clear()´ are removed
and replaced with a relevent equivalent.

The included list of words is sourced from [Mieliestronk's list of more than
58 000 English words](https://www.mieliestronk.com/wordlist.html).
